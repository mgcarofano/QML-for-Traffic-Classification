# QML for Traffic Classification
Repository per la tesi magistrale di **Mario Gabriele Carofano**, Studente dell'Università degli Studi di Napoli "Federico II", Corso di Laurea Magistrale in Informatica. Il progetto esplora la classificazione del traffico mobile cifrato tramite approcci QML, confrontando rappresentazioni FlowPic e MIMETIC su flussi bidirezionali, con estrazione di feature temporali e di protocollo da finestre a lunghezza fissa.
